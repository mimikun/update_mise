# update_mise

## Requirements

- [fisher](https://github.com/jorgebucaran/fisher)
- [mise](https://github.com/jdx/mise)

## Install

### Bash

```shell
git clone https://github.com/mimikun/update_mise.git
cd update_mise
sudo make install
```

### fish

```shell
fisher install mimikun/update_mise
```
